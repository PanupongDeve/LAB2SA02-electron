ประโยชน์

Electron เป็น framework ที่เอาไว้ทำ desktop application ด้วยเพียงแค่ การเขียนเว็บโดยการใช้ HTML, JavaScript, และ CSS ที่สำคัญเขียนทีเดียวทำ cross platform ได้เลยทันที ไส้ในของ Electron ก็คือ Chromium Electron จะแบ่งตัวเองเป็น 2 ส่วนหลักๆคือ Main process และ Renderer process
โดย Main process ให้นึกถึงว่ามันคือ Browser ตัวนึงที่จะคอยดูแล Renderer process ทุกตัว และ Renderer process ให้นึกว่ามันคือหน้าเว็บหน้าหนึ่งที่เราเปิดบน Chrome ในนั้นเราก็เขียน HTML ให้แสดงหน้า UI ตามที่เราอยากจะแสดงขึ้นมาได้

ข้อดี

- สามารถเขียน Build ครั้งเดียวแล้วลง ได้ 3 OS
- สามารถใช้ความรู้เดิมในการเขียน web เช่น java css สามารถนำความรู้เดิมมาต่อยอดได้เลย
- ง่ายในการใช้งานสำหรับผู้เริ่มต้นในการเขียน web

ข้อเสีย

- เป็นเทคโนโลยีที่ใหม่ มีเครื่องมือใหม่ที่ต้องศึกษาและติดตั้งเพิ่มเติม
- เป็นซอฟแวร์ที่ open source สามารถมี BUG เกิดขึ้นได้
